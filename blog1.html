My Blog

Summary: A personal story blog that highlights your transition.

Switching from Core Java to MERN Stack has been one of the most exciting learning journeys of my career. I started my career as a Java Developer, where I learned strong programming fundamentals, debugging, and backend problem-solving.

After a few years of break to take care of my family, I decided to restart my career in web development — an area that’s continuously evolving and full of opportunities. I chose the MERN stack (MongoDB, Express.js, React, Node.js) because it allows me to build complete web applications using one language: JavaScript

Every day, I’m learning something new — from handling APIs to designing responsive user interfaces. This journey is not just about learning new tools, but about rebuilding confidence and adapting to modern technologies.

My advice to anyone restarting their career: start small, stay consistent, and keep building projects.
